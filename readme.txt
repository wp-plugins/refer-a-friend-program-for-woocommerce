== Installation ==

1. Visit [StoreYa's website](http://www.storeya.com/public/referafriend/) and sign up for a Free account.

2. Create your referral campaign, set the offer and setup the refer a friend pop ups.

3. Copy the refer a friend 'Secret key' displayed at step no.6.

4. Unzip and Upload file storeya-refer-a-friend.php to the /wp-content/plugins/ directory 

5. Activate the plugin through the 'Plugins' menu in WordPress using your secret key.
